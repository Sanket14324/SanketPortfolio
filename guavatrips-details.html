Guava Trips - AI-Powered Travel Recommendation System

Project Information

  • Category: Full Stack Web Application
  • Client: Guavatrips.com, Pune
  • Project Name: Guava Trips - Travel Recommendation Platform
  • Project Date: 2024 - Present
  • Live URL: guavatrips.com

Built a comprehensive data-driven travel recommendation system using Spring Boot microservices with AI integration. Deployed on AWS with S3 integration for file storage and contributed to both user and admin modules.

Key Features:

  • AI-Powered Recommendations: Intelligent travel suggestions based on user preferences
  • User Portal: Complete travel planning and booking interface
  • Admin Dashboard: Comprehensive management system for travel packages
  • Data Analytics: Advanced analytics for travel trends and user behavior
  • File Management: Secure document and image handling

Technical Implementation:

  • Backend: Java Spring Boot with microservices architecture
  • Database: MySQL for robust data management
  • Cloud Platform: AWS deployment with scalable infrastructure
  • Storage: AWS S3 for file storage and management
  • AI Integration: Machine learning algorithms for personalized recommendations
  • API Design: RESTful APIs for seamless frontend integration

  • Developed both user and admin portal functionalities
  • Integrated AWS S3 for scalable file storage
