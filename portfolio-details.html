DocShare - CA Client Communication Platform

Project information

  • Category: Full Stack Web Application
  • Client: Personal Project
  • Project Name: DocShare - CA Client Communication Platform
  • Project date: 2024
  • Technology Stack: Spring Boot, MySQL, AWS S3, Thymeleaf

DocShare is a comprehensive web application facilitating secure document exchange and communication between Chartered Accountants (CAs) and their clients through a structured ticketing system.

Key Features:

  • Role-Based Authentication: Secure user registration/login with separate interfaces for CA admins and clients
  • Ticket Management: Complete ticketing workflow allowing clients to raise queries and CAs to track, respond, and manage ticket status
  • Document Management: AWS S3 integration for secure cloud storage with file upload, download, and deletion capabilities
  • Real-time Communication: Interactive comment system for discussions within each ticket
  • Admin Dashboard: Comprehensive admin panel for user management, ticket oversight, and system analytics

Technical Implementation:

  • Backend: Spring Boot 3.1.5, Spring Security 6, Spring Data JPA
  • Frontend: Thymeleaf templating, HTML5, CSS3, JavaScript
  • Database: MySQL with JPA/Hibernate ORM
  • Cloud Storage: AWS S3 for scalable file management
  • Security: JWT-based authentication with comprehensive validation
